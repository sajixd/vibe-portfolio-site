# AGENTS.md

## Project Overview

This project is a portfolio website built with [Astro](https://astro.build/). The workspace contains the following structure:

```
astro.config.mjs
package.json
README.md
src/
  env.d.ts
  components/
    Footer.astro
    Header.astro
  layouts/
    BaseLayout.astro
  pages/
    about.astro
    index.astro
    projects.astro
  styles/
    global.css
```

## Directory Structure

- **astro.config.mjs**: Astro configuration file.
- **package.json**: Project dependencies and scripts.
- **README.md**: Project overview and instructions.
- **src/**: Source files for the site.
  - **env.d.ts**: TypeScript environment definitions.
  - **components/**: Reusable UI components (Footer, Header).
  - **layouts/**: Layout templates (BaseLayout).
  - **pages/**: Site pages (about, index, projects).
  - **styles/**: Global CSS styles.

## Development

- **Start Dev Server**: `npm run dev`
- **Default Shell**: bash
- **OS**: Linux

## Agents & Automation

This workspace is managed and documented by GitHub Copilot (GPT-4.1), an AI agent that assists with coding, documentation, and automation tasks. The agent can:

- Generate and edit code files
- Document project structure and usage
- Run terminal commands
- Install dependencies
- Assist with debugging and error resolution

### Agent Capabilities

- **Code Generation**: Create new components, layouts, and pages.
- **Documentation**: Maintain project documentation files.
- **Automation**: Run scripts and manage dependencies.
- **Error Handling**: Analyze and resolve errors in code and configuration.

### Usage

To interact with the agent, simply describe your request (e.g., "Add a new page", "Document the project", "Fix errors"). The agent will perform the necessary actions and update the workspace accordingly.
